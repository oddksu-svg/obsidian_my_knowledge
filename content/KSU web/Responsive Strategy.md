# Responsive Strategy

The layout is designed to be mobile-first and fluid across breakpoints.

## Tablet (`768px - 1024px`)
- Shift from 3-column to 2-column grid.
- News and Admissions side-by-side.
- Academic Programs full width below.

## Mobile (`< 768px`)
- Header collapses into Hamburger Menu.
- 1-column vertical stack (News -> Admissions -> Faculties).
- Hero section text scales down for readability.
