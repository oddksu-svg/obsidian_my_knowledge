# Design System

The KSU Design System ensures consistency across the platform by adhering to institutional brand guidelines.

## Color Palette
| Category | Color Name | Hex Code | Usage |
| --- | --- | --- | --- |
| Primary | KSU Navy | `#003366` | Header, Hero, Text |
| Primary | KSU Light Blue | `#4AA8D8` | Buttons, Highlights |
| Accent | Admissions Green | `#2E8B57` | Admissions Section |
| Background | Soft BG | `#F4F7F9` | Section separation |

## Typography
- **Headings (Thai/EN):** `Prompt` or `Kanit` (Modern, geometric)
- **Body Text:** `Sarabun` (Official Thai) or `Inter` (Professional EN)

## Grid & Containers
- **Max Width:** `1280px`
- **Columns:** 12-column CSS Grid / Flexbox
